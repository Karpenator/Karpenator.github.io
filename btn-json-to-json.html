📦 DayZ Editor to CacheSpawner

Конвертирует JSON из DayZ Editor в формат CacheSpawner.

📖 Как использовать:

1. Вставьте JSON из DayZ Editor (m_SceneObjects)
2. Нажмите "Convert"
3. Скопируйте результат в конфиг CacheSpawner

Входной формат: {"m_SceneObjects": [[classname, [x,y,z], [rx,ry,rz]], ...]}
Выходной формат: CacheList с Classname, Coords, ItemsInside
