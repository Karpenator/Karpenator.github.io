🏪 Trader Config Creator

Генератор конфигов для различных трейдеров. Поддерживает: Expansion Trader, Dr. Jones, Trader Plus, Ultima.

📖 Как использовать:

1. Вставьте список classnames (каждый с новой строки) или загрузите файл
2. Выберите тип трейдера из вкладок
3. Настройте параметры (цены, лимиты)
4. Нажмите "Generate Config"
5. Скачайте полученный конфиг

Пример classnames:
Ammo_9x19
MedKit
BandageDressing

Supports .txt files with one classname per line
