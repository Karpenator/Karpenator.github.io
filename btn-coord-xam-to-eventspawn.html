📍 dayz.xam.nu Coordinate to XML

Конвертирует координаты с dayz.xam.nu в XML формат для event spawn.

📖 Как использовать:

1. Скопируйте координаты с dayz.xam.nu
2. Вставьте координаты в формате: x, z (каждая пара с новой строки)
3. Нажмите "Convert to XML Format"
4. Скопируйте результат

Пример входа:
1234.56, 7890.12
2345.67, 8901.23

Пример выхода:
<pos x="1234.56" z="7890.12" />
<pos x="2345.67" z="8901.23" />
