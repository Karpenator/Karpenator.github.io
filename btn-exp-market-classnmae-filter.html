🔍 EXP Market Classnames Filter

Фильтрует JSON EXP Market по списку classnames и добавляет стандартные параметры.

📖 Как использовать:

1. Введите список classnames для фильтрации
2. Вставьте JSON EXP Market
3. Нажмите "Изменить JSON"
4. Скопируйте отфильтрованный результат

Пример classnames:
Shrimp
WalleyePollock

Результат: JSON только с указанными ClassName + стандартные параметры
