🚙 Vehicle Market to Events XML

Конвертирует JSON EXP Market с автомобилями в XML формат для event spawn.

📖 Как использовать:

1. Вставьте JSON EXP Market с автомобилями
2. (Опционально) Введите фильтры classnames для выборки
3. Нажмите "Convert to DayZ XML Format"
4. Скопируйте результат в events.xml

Входной формат: [{"ClassName": "CarName", "SpawnAttachments": [...]}]
Выходной формат: XML с <type name="..."> и <attachments>
