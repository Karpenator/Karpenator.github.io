📥 Extract Classnames from EXP

Извлекает список classnames из JSON конфига Expansion Trader и фильтрует по максимальной цене. Полезно для создания списков товаров для других трейдеров.

📖 Как использовать:

1. Вставьте JSON файл конфига Expansion Trader
2. Укажите максимальную цену для фильтрации
3. Нажмите "Фильтровать"
4. Скопируйте полученный список classnames
