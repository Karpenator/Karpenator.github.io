📋 Генератор JSON для SLS

Генерирует JSON конфигурацию для Simple Loot Spawner (SLS) из списка classnames предметов. Укажите шанс спавна и получите готовый конфиг.

📖 Как использовать:

1. Введите названия предметов (каждый с новой строки)
2. Укажите шанс спавна (например, 0.53 = 53%)
3. Нажмите "Генерировать"
4. Скопируйте результат в конфиг SLS
