🚗 Vehicle Classnames to Events XML

Создаёт XML для event spawn с attachments из списка classnames автомобиля и его частей.

📖 Как использовать:

1. Введите список classnames автомобиля и его частей
2. Нажмите "Convert to DayZ events"
3. Скопируйте результат в events.xml

Пример входа:
IJ_2140_SL
IJ_2140_SL_wheel
IJ_2140_SL_doors_driver
IJ_2140_SL_doors_hood

Пример выхода:
<type name="IJ_2140_SL">
<attachments chance="1.00"><item name="CarRadiator" chance="0.10" /></attachments>
<attachments chance="1.00"><item name="IJ_2140_SL_wheel" chance="0.40" /></attachments>
